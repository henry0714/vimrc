# vimrc
vim settings and plugins

# MANUALLY INSTALL
1. Check out from GitHub

        git clone https://github.com/henry0714/vimrc.git ~/.vim
        cd ~/.vim

2. Install ~/.vimrc

        cd ~ && ln -s ~/.vim/vimrc ~/.vimrc 

3. Install plugin by Vundle, and lunch Vim

        vim +BundleInstall
